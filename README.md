# Estudos Python

Este repositório é dedicado aos meus estudos de **Python para iniciantes**. Aqui, vou registrar exercícios, pequenos projetos e anotações enquanto aprendo os fundamentos da linguagem, incluindo:

- Sintaxe básica
- Variáveis e tipos de dados
- Estruturas de controle 
- Funções
- Listas, dicionários e outros tipos de coleções
- Manipulação de arquivos
- Introdução à programação orientada a objetos (OOP)

O objetivo é praticar todos os conceitos aprendidos e documentar meu progresso no aprendizado de Python.
